# Near ORM Project

A simple ORM for IndexedDB, inspired by Prisma.

## Packages

- [near-orm](packages/core) - The core ORM package, the OG `near-orm`

## Development

This is a monorepo using npm workspaces. To get started:

```bash
# Install dependencies
npm install

# Build all packages
npm run build

# Run tests
npm test
```
